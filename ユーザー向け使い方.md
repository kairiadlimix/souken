# 図面チェックAIシステム - ユーザー向け使い方

## 概要

このシステムは、設計事務所から提出される図面を自動的にチェックし、創建基準に基づいた指摘を行うWebアプリケーションです。

## 使い方

### 方法1: Webアプリケーション（推奨）

ブラウザから簡単に使えるWebアプリケーションです。

#### ローカルで起動する場合

1. **プロジェクトディレクトリに移動**
   ```bash
   cd /Users/kairi.oshima/souken
   ```

2. **依存パッケージのインストール**（初回のみ）
   ```bash
   pip3 install -r requirements.txt
   ```

3. **Webアプリケーションを起動**
   ```bash
   streamlit run app.py
   ```

4. **ブラウザでアクセス**
   - 自動的にブラウザが開きます
   - または、表示されたURL（通常は http://localhost:8501）にアクセス

5. **図面をチェック**
   - PDFファイルをアップロード
   - 「チェック実行」ボタンをクリック
   - 結果を確認

#### クラウドで使う場合（今後実装予定）

1. ブラウザでURLにアクセス
2. PDFファイルをアップロード
3. チェック結果を確認

### 方法2: コマンドライン

技術者向けのコマンドライン実行方法です。

```bash
# プロジェクトディレクトリに移動
cd /Users/kairi.oshima/souken

# 図面をチェック
python3 -m src.main 図面ファイル.pdf

# JSON形式で結果を出力
python3 -m src.main 図面ファイル.pdf --format json --output result.json
```

## チェック項目

### 必須記載事項
- 図面番号
- 図面名
- 縮尺
- 作成日
- 作成者

### 創建特有項目
- 外断熱仕様
- 第一種換気システム
- 釘ピッチ（150mm以下）
- 隠蔽部分の施工方法

## よくある質問

### Q: どのようなPDFファイルに対応していますか？
A: CADから出力されたPDF（ベクター形式）に対応しています。スキャンされたPDF（画像形式）は今後の対応予定です。

### Q: チェックにどのくらい時間がかかりますか？
A: 図面のサイズによりますが、通常は数秒から数十秒です。

### Q: 結果を保存できますか？
A: Webアプリケーションでは、結果をテキスト形式でダウンロードできます。

### Q: エラーが発生した場合は？
A: エラーメッセージを確認し、以下を試してください：
- PDFファイルが正しく読み込めるか確認
- ファイルサイズが大きすぎないか確認（推奨: 50MB以下）
- ブラウザを再読み込み

## トラブルシューティング

### Streamlitが起動しない

```bash
# Streamlitがインストールされているか確認
pip3 show streamlit

# インストールされていない場合
pip3 install streamlit
```

### ポートが既に使用されている

```bash
# 別のポートで起動
streamlit run app.py --server.port 8502
```

### モジュールが見つからないエラー

```bash
# プロジェクトルートから実行していることを確認
cd /Users/kairi.oshima/souken
streamlit run app.py
```

## サポート

問題が発生した場合は、以下を確認してください：
1. エラーメッセージの内容
2. 使用したPDFファイルの情報
3. 実行環境の情報（OS、Pythonバージョンなど）
